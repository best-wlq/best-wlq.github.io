测试集合

发表于 2022-02-09 分类于 杂谈

简单说明

  • 网上关于博客搭建的教程已经很多，建议bilibili+关键词（hexo博客搭建教程）搜索得到详细的搭建流程。

  • 对于用到的几个工具的使用逻辑说明一下。

    工具 功能
    Node.js+npm 最底层的工具，方便后续的软件的安装和插件的管理
    Hexo 开源的博客框架，安装好它就可以得到一个基本的博客模板
    Next 基于hexo博客框架的一种博客主题，使用广泛，简约自然
    git 版本控制工具，这里主要是为了方便部署到远端
    gitee/github 云端的代码仓库，即免费的服务器，用于托管本地的博客到云端，使得别人能够通过互联网访问你的博客
    hexo-xxx-xx hexo的插件，用于实现博客的拓展功能，比如使用数学公式等

参考

  • Hexo + Git 搭建免费的个人博客
  • Hexo配置NexT主题

几种方案比较

  1. github虽然可以自动更新页面，但是访问不稳定。
  2. gitee(码云)虽然访问稳定，但是不自动更新页面，自动更新服务需要付费。
  3. 买域名+自建服务器，解决了上述问题，但是租用ip和vps每年有相对较高的开销。
